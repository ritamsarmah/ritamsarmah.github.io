Quickly Run Commands


Bring the power of the terminal to the Finder and/or menu bar with custom commands. Here are some useful examples to get you started.


Menu Bar Commands

Menu bar commands run in the user's home directory (~/)


Advanced Mac Controls

Disable Sleep for an Hour (in Seconds)

caffeinate -t 3600

Clear RAM Cache

sudo purge

Networks and Remote Access

Copy Wi-Fi IP Address

ipconfig getifaddr en0 | tr -d '\n' | pbcopy

Connect to VPN and Login via SSH

networksetup -connectpppoeservice "My VPN";
ssh username@server.domain.com

Finder Commands

Finder commands run in the current directory


Working with Folders

Open Folder in Xcode

open -a "Xcode" .

Move Folder to Desktop

mv . "~/Desktop"

Git Operations

Fetch Branches

git fetch origin

Delete Last Pushed Commit

git reset --hard HEAD~1;
git push origin HEAD --force

Other Useful Hints

  • Use Option + Return to add new lines while editing commands

  • Combine commands with semicolons (e.g. command1 args; command2 args)

  • Use exit to automatically close the terminal window after running
