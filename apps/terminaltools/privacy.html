Privacy Policy



Terminal Tools never collects any personal data or identifiable information from you. Personalized app features, like preferences and commands, are stored locally on your computer.


Permissions


Terminal Tools asks for two types of macOS permissions to properly work. Finder Extension permissions are required for displaying the toolbar button and right-click options in Finder. Automation permissions are required to interact with other terminal applications and run commands. Both can be configured in System Preferences or in Terminal Tools (General > Request Permissions).



If you have any additional questions about Terminal Tools or privacy, you may contact the developer via email.
